5. Using the db

In order to use the database you need to have a PostgreSQL database instance running somewhere in your computer (if you do it locally) or in the cloud (e.g., in AWS, GC, etc.).

Note

We recommend you get a graphical tool to manage and develop your PostgreSQL databases. The PostgreSQL comes with PgAdmin, which is quite basic. We suggest you get a more advanced tool that can handle many database types, such as DBeaver or Pycharm.

  1. If you don’t have PostgreSQL running in your computer you need to install it first. You can download it here

  2. Download the database (see the Download section) and unzip it. The .tar file you unzipped contains the database you are about to install

  3. Open DBeaver

  1. Create a new connection to your PostgreSQL instance: Database -> New Database Connection -> PostgreSQL

  2. In the Database Navigator go to your connection (typically called “postgres” by default), and expand it down to Databases -> postgres

  3. Right click on “postgres”, go to Tools -> Restore

  4. Inside the “input” section, in the “Backup file” box choose the .tar file you got from us

  5. Click “Start”
